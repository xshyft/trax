# traxcli

`traxcli` is the operator and worker CLI for TRAX.

## Responsibilities

- interactive and direct control of `traxctrl` APIs;
- workflow template creation helpers;
- standalone executor runtime;
- standalone submitter runtime;
- watch and inspection helpers.

## Paths

- binary entrypoint: `cmd/traxcli`
- cobra command package: `cmd/agora/clis/traxcli`
- runtime package: `pkg/clis/traxcli`
