# Agent Instructions

This repository keeps an **agent-maintained wiki** as a source of truth for TRAX concepts,
architecture, components, and operational behavior.

Before substantial work, read [`wiki/`](wiki/), starting at [`wiki/index.md`](wiki/index.md).
As you work, update the wiki and keep [`wiki/index.md`](wiki/index.md) and
[`wiki/glossary.md`](wiki/glossary.md) current.

Git commits and pushes still require a clear human command.
